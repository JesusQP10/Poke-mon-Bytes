# Sprites de Items

Coloca aquí los sprites de items del juego.

## Sprites necesarios:

### 1. pokeball.png
- **Tamaño:** 8x8 px o 16x16 px
- **Descripción:** Pokeball estándar
- **Uso:** Representa las pokeballs en la mesa del laboratorio (antes de elegir starter)
- **Ubicación:** elm_lab.json
- **Estilo:** Pokeball roja y blanca, vista desde arriba

## Formato:

- **Tipo:** PNG con transparencia
- **Tamaño:** 8x8 px (pequeña) o 16x16 px (más visible)
- **Vista:** Cenital (desde arriba)
- **Paleta:** Rojo (#FF0000), Blanco (#FFFFFF), Negro (#000000) para el contorno
